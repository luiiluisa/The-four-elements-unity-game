# The Four Elements

The Four Elements is a 2D game developed in Unity.
In this game, the player controls a character that collects cherries while progressing through four different levels. Each level represents one of the four classical elements: fire, water, earth and air.

## Project Structure
Documentation – contains the project documentation.  
Game/GameLook – contains a video showing how the game looks.  
Game/Scripts – contains the main C# scripts.
